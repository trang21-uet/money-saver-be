# Product Overview

Banana is a comprehensive expense management application that helps users track their finances, manage accounts, and monitor spending patterns.

## Core Features

- User authentication via Google OAuth2 and JWT
- Multi-account management (cash, bank, credit card)
- Transaction tracking with categories
- Budget management and monitoring
- Savings goals tracking
- Group expense sharing
- OCR receipt scanning and classification
- Financial statistics and reports
- Data synchronization and backup

## Target Users

Individuals and groups who need to track personal finances, manage multiple accounts, and gain insights into their spending habits.

## Architecture

Backend REST API built with Spring Boot serving data to mobile clients. The system uses PostgreSQL for persistent storage and Redis for caching and session management.
